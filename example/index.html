SetCursorLine Example

Example

SetCursorLine only

With SetNumber and Autosize

Usage

Load jQuery and SetCursorLine (js)

          ;; e.g. CDN
          <script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>

          ;; SetCursorLine
          <script type="text/javascript" src="../path/to/set-cursor-line.min.js"></script>

Put textarea

          <body>
            <textarea id="editor"></textarea>
          </body>

Apply setCursorLine

          <script>
            var ready = function() {
              $("#editor")
                .setCursorLine({ // set-cursor-line
                  cursorLineColor:   "silver"
                })
                .focus();
            };
            $(document).ready(ready);
          </script>
